How to get your Square credentials (step by step)
  1. Go to the Square Developer Dashboard and sign in with the Square account that should receive the money.
  2. Click + (Create app) — name it e.g. YoutilityKnock — or open an existing app.
  3. At the top of the app, set the environment toggle: Sandbox to test, Production to take real payments. Set the matching Environment dropdown above.
  4. Open the app's Credentials page:
    • Copy the Application ID (prod starts with sq0idp-, sandbox with sandbox-sq0idb-) → paste into Application ID.
    • Copy the Access token (production or sandbox) → paste into Access token. Keep it secret, like a password.
  5. Open Locations in the left menu, copy the Location ID (starts with L) of the location that should receive payments → paste into Location ID.
  6. Open Webhooks → Subscriptions, click Add endpoint:
    • URL: https://youtilityknock.web.app/square/webhook
    • API version: the latest available.
    • Events: enable payment.created and payment.updated.
    • Save, then copy the endpoint's Signature key → paste into Webhook signature key.
  7. Click Save Square keys, set Active payment provider to Square, and click Use this provider.
